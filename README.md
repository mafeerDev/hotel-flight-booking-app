# Hotel Flight Booking App

Mobile application developed as an academic project focused on hotel and flight reservations.

## Features

* Search cities
* Browse hotels
* Browse flights
* Reservation simulation flow
* Mobile friendly interface
* Navigation between screens

## Tech Stack

* React Native
* JavaScript
* Firebase
* Expo

## Purpose

This project was created to practice mobile development, UI design and app navigation using React Native.
